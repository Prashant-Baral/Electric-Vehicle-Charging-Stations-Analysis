## Overview

This project analyzes EV charging infrastructure using data from the U.S. Department of Energy's Alternative Fuels Data Center. The analysis examines charger types (Level 1, Level 2, DC Fast), geographic distribution, facility types, access restrictions, and growth trends.

### Key Findings

- California, New York, Florida, and Texas lead in station deployment
- Level 2 chargers dominate (58.21K), while DC Fast charging remains limited (42.60K)
- ~90% of stations are publicly accessible
- Retail and hospitality facilities host the majority of stations
- Strong urban clustering with significant rural gaps
